Python 包装用户指南¶

欢迎来到 Python 包装用户指南，这是一个教程和参考资料的集合，帮助你用现代工具分发和安装 Python 包。

本指南由 Python Packaging Authority 在 GitHub 上维护。我们很乐意接受任何 贡献和反馈。 😊

开始¶

我们的 教程 部分介绍了在 Python 开发生态系统中工作的基本工具和概念：

  • To get an overview of the flow used to publish your code, see the packaging flow

  • 要学习如何安装软件包，请参阅 安装软件包的教程

  • 要学习如何在版本控制项目中管理依赖关系，请参见 管理应用程序依赖关系的教程

  • 要学习如何打包和发布您的项目，请参阅 关于打包和发布的教程

  • 要获得对 Python 库和应用程序打包选项的概述，请参阅 Python 打包概述

了解更多¶

除了我们的 教程 ，本指南还有其他一些资源：

  • 指南 部分用于排练，例如 用 Linux 软件包管理器安装 pip/setuptools/wheel 或 Packaging binary extensions。

  • 讨论 部分，用于深入参考诸如 部署 Python 应用程序 或 pip 与 easy_install 等主题。

  • PyPA 规范 部分，用于包装互操作性规范。

此外，还有一个 其他项目 的列表，由 Python Packaging Authority 的成员维护。
